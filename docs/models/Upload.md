## Upload

Metadata for a stored file uploaded by a user.

### Key Relations

- Belongs to `User`

### Notable Constants

- `PUBLIC_UPLOAD` and `PRIVATE_UPLOAD` visibility flags.

### Typical Fields

- `id`, `user_id`, `slug`, `title`, `description`, `is_private`, `original_name`, `stored_name`, `mime_type`, `size`, `path`
